# databricks-apache-spark-developer
# 📚 Estudos para Certificação Apache Spark Developer (Databricks)

Este repositório foi criado com o objetivo de organizar e documentar meus estudos para a certificação **Apache Spark Developer** da **Databricks**.

Aqui você encontrará anotações, exemplos práticos, exercícios e materiais de apoio que estou utilizando ao longo da minha preparação.

---

## 🎯 Objetivo

Consolidar conhecimentos em **Apache Spark** e estar preparado para obter a certificação, com foco em:

* Processamento distribuído de dados
* Manipulação de dados com Spark
* Uso de DataFrames e Spark SQL
* Otimização de performance
* Boas práticas no uso do Spark

---

## 🧠 Conteúdos Estudados

* Fundamentos do Apache Spark
* Arquitetura do Spark
* Spark DataFrames
* Spark SQL
* Transformations vs Actions
* Lazy Evaluation
* Particionamento e paralelismo
* Otimização de queries
* Leitura e escrita de dados (Parquet, JSON, CSV, etc.)
* Introdução ao Delta Lake
